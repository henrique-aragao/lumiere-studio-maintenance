## 📖 Sobre o Projeto

O **Lumière Studio** é uma página de manutenção desenvolvida para representar um estúdio de fotografia moderno e artístico.

O objetivo foi criar uma experiência elegante durante o período de indisponibilidade do site, utilizando uma identidade visual inspirada no universo da fotografia, com tipografia sofisticada, cores suaves e um layout limpo.

O projeto foi desenvolvido utilizando apenas **HTML5** e **CSS3**, sem bibliotecas ou frameworks.

## 📱 Responsividade

O projeto foi desenvolvido seguindo a abordagem **Desktop First**.

### Desktop
- Layout dividido em duas colunas
- Fotografia artística ao lado do conteúdo

### Mobile
- Layout reorganizado verticalmente
- Imagem otimizada para telas menores
- Botão adaptado para melhor usabilidade
- Ajustes de tipografia e espaçamento
